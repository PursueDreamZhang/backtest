# 趋势跟踪策略逻辑说明

本文档用于说明项目当前默认趋势跟踪策略的交易逻辑，重点描述策略的买入条件、卖出条件、仓位控制方式以及整体思路，便于从业务角度理解策略行为。

## 策略概览

该策略的核心思路不是简单追涨，而是尝试在强势股完成一轮拉升之后，等待其回调到关键支撑区域，再观察是否出现缩量企稳和重新向上突破的信号，随后以受控仓位介入。如果买入后走势不够强，则尽快退出；如果走势延续，则尽量让利润继续扩张，但在趋势转弱时及时止盈离场。

从交易风格上看，这是一套偏向“强势股二次启动捕捉”的趋势策略。

## 买入逻辑

策略的买入逻辑分成多个阶段，只有全部条件同时满足时才会触发买入。

### 1. 强势股识别

策略会先观察最近一段时间内的价格表现，判断该股票是否曾经出现过明显上涨。默认规则是：最近 30 个交易日内，区间涨幅达到或超过 50%，则将该股票视为已经完成过一轮强势拉升。

这一步的目的，是先把目标限定在近期具有明显攻击性的股票上，而不是从所有普通波动个股中随机寻找机会。

### 2. 从高点回落

被识别为强势股之后，策略不会立即追入，而是继续等待其从近期高点出现一定幅度的回调。默认要求是：当前价格相对前期强势阶段高点，至少回落 5%。

这一步的目的，是避免在短线最亢奋的位置直接追高，希望等到价格回到更有性价比的位置再考虑介入。

### 3. 靠近支撑位

在回落之后，策略还会判断当前价格是否已经接近关键支撑区域。默认会同时参考以下几类支撑：

- 中短期均线支撑
- 主趋势均线支撑
- 近期箱体下沿支撑

只要当前价格距离这些支撑中的任意一个足够近，就认为价格已经回到了可关注的支撑区域。默认要求是：当前价格与最近支撑位的偏离不超过 5%。

这一步的作用，是尽量让买点出现在回调结束、支撑附近，而不是仍处在明显高位或下跌途中。

### 4. 缩量调整

策略会继续判断回调过程中的成交量是否出现明显收缩。默认规则是：短期平均成交量明显低于中期平均成交量。

这意味着在价格调整时，市场抛压正在减弱，筹码换手速度放慢，情绪比前期上涨阶段更冷静。策略将这种状态视为后续重新启动的重要辅助条件。

### 5. 再次向上突破

在确认强势背景、回调幅度、支撑位置和缩量状态后，策略还需要一个重新转强的触发信号。默认规则是：当日盘中价格相对前一交易日收盘价上涨达到 5% 或以上，即视为突破触发。

这里强调的是“盘中突破”，不是必须等到收盘确认。这意味着策略更注重重新启动的即时性，也因此会比只看收盘信号更敏感、更主动。

### 6. 买入触发

只有在以下条件同时满足时，策略才会发出买入指令：

- 近期曾经是强势股
- 当前已经从高点回落
- 当前价格接近关键支撑位
- 调整过程出现缩量
- 当天盘中重新向上突破

这套组合条件的本质，是希望抓住“强势股回调后的二次启动”。

## 仓位控制

策略即使确认买点，也不会直接满仓介入，而是通过风险和资金上限两套规则共同约束实际买入数量。

### 1. 先计算结构止损位

在准备买入时，策略会先估算一个结构止损位置。这个位置通常取自以下几种参考中的更保守者：

- 主趋势均线附近
- 近期箱体低点附近
- 买入价格下方一个较小的保护空间

这个止损位代表策略对当前趋势结构的容忍边界，一旦跌破，说明原先的买入逻辑很可能已经失效。

### 2. 根据单笔风险控制仓位

策略默认会限制单笔交易最大可承受亏损占总资产的比例。也就是说，会先估算“每买一股最多可能亏多少钱”，再根据账户净值反推出合理持仓股数。

这种做法的意义在于：不是按感觉决定买多少，而是先定风险，再定仓位。

### 3. 根据资金占用上限控制仓位

除了风险控制之外，策略还会限制单只股票最多占用多少可用资金。默认情况下，会给单票设置一个最高仓位比例，防止在单一标的上过度集中。

### 4. 取两者中更小的结果

最终实际买入股数，会在“风险约束算出的仓位”和“资金上限算出的仓位”中取更小值。

这样做的结果是：

- 即使止损距离很近，也不会因为理论风险较小而过度放大仓位
- 即使账户资金充裕，也不会因为一只票看起来很强而下过重仓位

## 卖出逻辑

买入之后，策略会持续监控持仓表现，一旦满足任一卖出条件，就执行离场。卖出条件按风险优先、效率优先、趋势优先的思路设计。

### 1. 结构止损

如果价格跌破买入时设定的结构止损位，策略会直接卖出。

这表示市场走势已经破坏了原先“支撑位附近企稳后继续上行”的判断基础，继续持有的理由明显减弱。

### 2. 硬止损

除了结构止损，策略还设置了一个固定幅度的兜底止损。默认情况下，如果持仓浮亏达到 10%，也会直接离场。

这是一层额外保护，用来防止在特殊情况下结构判断失效，导致亏损持续扩大。

### 3. 持有期考核

策略认为，既然买入的是“二次启动”机会，那么买入后短时间内就应该体现出一定强度。如果持有达到预设天数之后，收益仍然达不到最低目标，则说明这次启动力度不足，策略会主动退出。

默认理解是：如果买入后短期内既没有迅速走强，也没有形成足够利润，那么宁可放弃，也不拖成低效率持仓。

### 4. 盈利后的趋势转弱止盈

如果持仓已经产生浮盈，策略会允许利润继续扩张，但不会无限持有。一旦价格跌破用于跟踪短期强势的均线，就视为趋势开始钝化，策略会选择止盈离场。

这一步的逻辑是：在行情向上时尽量持有，在趋势不再流畅时及时兑现。

### 5. 单日异常下跌

如果持仓期间出现异常的单日大跌，策略也会立即卖出。

这种规则主要用于应对突发性反向波动。即使其他趋势结构暂时还没有完全破坏，单日过大的下跌也意味着风险显著提升，不再适合继续死扛。

## 执行顺序理解

从整体执行顺序上看，策略可以概括为：

1. 先找出最近曾大涨过的强势股
2. 等其从高点回调到支撑区域
3. 观察回调过程中是否缩量
4. 等待重新向上突破作为入场触发
5. 买入后先控制风险，再观察是否快速走强
6. 如果走势不及预期，尽快退出
7. 如果走势持续向上，则采用跟随式止盈方式保护利润

## 策略特点

这套策略有几个比较鲜明的特征：

- 它不是抄底策略，而是强势延续策略
- 它不是纯追涨策略，而是等待回调后的再启动
- 它强调支撑、缩量和突破三个条件的组合
- 它对买入后的表现有较高要求，不愿意长期持有弱势仓位
- 它用仓位控制和多层退出机制来限制单笔交易风险

## 适用理解

从理解上看，这套策略更适合用来研究如下问题：

- 强势股在回调后是否存在较高概率的二次启动机会
- 缩量回调加支撑企稳是否能够提升突破后的成功率
- 短持有期考核和趋势止盈结合，是否能改善收益回撤比

它不一定适合所有市场环境。在趋势不明显、热点切换过快或个股经常冲高回落的阶段，策略表现可能会明显波动。因此在实际使用时，除了看单次回测结果，也应关注不同年份、不同市场阶段下的稳定性。
