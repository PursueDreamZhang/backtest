# 上班族日线波段交易规则

## 一、适用边界

这份手册适用于下面这类交易者：

- 白天要上班，不能高频盯盘
- 每天最多稳定看 `开盘 / 午间 / 收盘` 三次
- 交易级别以 `日线` 为主
- 交易对象以 `行业ETF + 板块龙头` 为主
- 风格偏进取，但不做纯超短和打板

这份手册不适用于：

- 盘中频繁交易
- 长期死拿不动的价值投资
- 明显下降通道里的抄底博反转
- 追求当天买当天大赚的超短打法

## 二、核心定位

这套方法的目标不是抓最极致的买点，而是用你现有的时间条件，稳定抓到日线级趋势中的一段“鱼身”。

核心思路只有三句：

1. `ETF管方向，龙头抓弹性`
2. `先有趋势，后等回档，不乱追高`
3. `错了先走，做对再加`

默认持有周期：

- 短一点：`3到5个交易日`
- 中等：`1到3周`
- 最长：`4周左右`

超过这个区间还在“磨”，通常说明效率不高，需要重新评估。

## 三、账户结构

你的风格设定为“偏进取版”，但仍保留基础生存线。

### 1. 账户分工

- `ETF仓位`：负责先上车、保留主线敞口、降低情绪波动
- `龙头仓位`：负责在主升段放大利润
- `机动仓`：永远保留 `10%到20%`，用于处理突发机会或防守

### 2. 总体原则

- 不是时时刻刻都要满仓
- 不是因为看好就无限加仓
- 不是ETF和龙头同时重到极致
- 重仓只能出现在“主线明确 + 趋势正常 + 结构没坏”的阶段

## 四、市场阶段判断

每天先判断当前属于哪一档市场，再决定ETF和龙头的重心。

### 1. 混沌期

特征：

- 主线不清，热点轮动快
- 板块此起彼伏，但持续性差
- 龙头经常今天强、明天弱
- 追涨容易被反杀

处理原则：

- 以ETF为主
- 龙头只小仓试错
- 看错就快撤

参考仓位：

- 总仓位：`20%到50%`
- ETF：`20%到40%`
- 龙头：`0%到20%`

### 2. 启动期

特征：

- 市场开始选出方向
- 板块ETF开始放量转强
- 龙头开始脱颖而出
- 第一次回档往往有参与价值

处理原则：

- ETF和龙头都可参与
- 先用ETF铺方向，再逐步加龙头

参考仓位：

- 总仓位：`40%到70%`
- ETF：`20%到30%`
- 龙头：`20%到40%`

### 3. 主升期

特征：

- 主线明确
- ETF和龙头共振上行
- 回档后容易再起
- 龙头强于板块，板块强于市场

处理原则：

- 龙头仓位可以明显高于ETF
- 重点赚趋势主升的钱
- 但仍保留机动仓，不写成长期满仓

参考仓位：

- 总仓位：`60%到90%`
- ETF：`10%到20%`
- 龙头：`40%到70%`
- 机动仓：`10%到20%`

### 4. 退潮期

特征：

- 龙头钝化、破位、冲高回落增多
- 板块内部分化加剧
- 利好刺激也带不动价格
- 市场更容易高开低走

处理原则：

- 先降龙头仓位
- 再决定ETF是否保留
- 如果板块整体也破位，ETF和龙头一起降

参考仓位：

- 总仓位：`0%到40%`
- ETF：`0%到20%`
- 龙头：`0%到20%`

## 五、ETF和龙头怎么切换

ETF和龙头不是二选一，而是不同阶段承担不同任务。

### 1. 什么时候优先ETF

优先ETF的场景：

- 主线刚露头，但你还看不清谁是真龙头
- 板块方向对，但个股分化很大
- 市场还在混沌到启动过渡阶段
- 你确认板块强，但不想承受个股的大波动

一句话：`方向先对，细节可以晚一点。`

### 2. 什么时候从ETF切到龙头

从ETF切向龙头的触发信号：

- 主线已经被市场确认
- 龙头成交活跃，辨识度高
- 龙头明显强于同板块其他个股
- 龙头第一次回档不破，或二次压紧后再起
- 板块ETF维持强势，不拖后腿

一句话：`先用ETF占位，再用龙头放大利润。`

### 3. 什么时候从龙头切回ETF

从龙头切回ETF的场景：

- 龙头已经在高位，波动过大
- 龙头钝化，但板块整体还没坏
- 你仍然看板块，但不想再承受个股大回撤
- 板块还有第二波预期，但龙头位置已经不舒服

一句话：`板块还能看，但个股已经不好拿。`

### 4. 什么时候ETF和龙头一起降仓

下面几种情况出现时，ETF和龙头都不能恋战：

- 板块整体破位
- 主线退潮
- 利好尽出，行情走到鱼尾
- 时间价值耗尽，反复走不顺
- 明显进入下降通道

一句话：`板块逻辑没了，ETF也不是护身符。`

## 六、三段式每日流程

你的交易日只分三段：`开盘确认`、`午间看结构`、`收盘做主决策`。

### 1. 开盘阶段：9:25-10:00

开盘只做确认，不做临时起意。

你只回答三个问题：

1. 今天市场是在延续昨天的强势，还是直接转弱
2. 持仓有没有异常低开、高开或突发利空
3. 自选标的有没有进入你昨晚写好的计划买点

允许的动作只有两种：

- `预设买点触发后的计划内买入`
- `明显破位或突发风险的应急卖出`

开盘禁止动作：

- 看见高开就追
- 盘中改剧本
- 因为怕错过，去买昨天没计划的票

### 2. 午间阶段：11:30-13:30

午间的任务不是多操作，而是看结构有没有变化。

重点看：

- 主线还在不在
- 持仓是否仍然符合原逻辑
- 计划买入的标的是不是走成了你预期的样子

午间只分三类结论：

- `继续持有`：趋势正常，不动
- `准备下午处理`：出现钝化、冲高回落、承接变差
- `放弃交易`：原本想买，但走势明显不像预期

### 3. 收盘阶段：14:50-15:10

这是你最重要的主决策时段。

收盘要完成四件事：

1. 判断市场现在属于 `混沌 / 启动 / 主升 / 退潮` 哪一档
2. 决定持仓是继续拿、减仓还是撤退
3. 决定明天重点看哪些ETF和哪些龙头
4. 写好明天的买点、止损位和仓位计划

动作优先级固定为：

1. 先处理风险
2. 再处理持仓
3. 最后才考虑新开仓

## 七、买入规则

只做三种买点，别扩展太多花样。

### 1. 第一次回档

定义：

- 主线已经明确
- 板块或个股完成第一段上涨
- 第一次明显回到关键支撑
- 跌不动后重新转强

常见支撑：

- `20天线`
- `40天线`
- `震荡区间下沿`
- `前低`

适用：

- ETF：非常适合
- 龙头：也适合，但要求更强的承接和转强动作

### 2. 二次压紧 / 双底

定义：

- 第一次回档后没有彻底走坏
- 再次回到支撑区
- 形成双底、二次压紧、回踩不破
- 再次出现转强信号

适用：

- ETF：可作为加仓点或低风险重上车点
- 龙头：适合趋势还在、但第一次没买够时参与

### 3. 启动第一阳线

定义：

- 长时间整理后
- 出现第一根明确启动阳线
- 板块和个股同步转强

注意：

- 这是唯一允许你稍微追一点的场景
- 但仓位要比回档买点更谨慎
- 止损必须更严格

### 4. ETF的买入要求

ETF可以比龙头更偏“确认后上车”，因为它波动更平滑。

满足下面几项即可考虑：

- 板块方向明确
- ETF结构没坏
- 回档到 `20天线 / 40天线 / 区间下沿`
- 缩量后重新转强

### 5. 龙头的买入要求

龙头的要求必须更高，因为你承担的是更高波动。

满足下面几项再考虑：

- 属于当前主线
- 辨识度明确
- 价格和成交都强于同板块其他个股
- 回档不破关键支撑
- 转强动作明确，不是弱反弹

### 6. 买入前必须写清楚的六个问题

每次开仓前，必须先写清楚：

1. 这笔是ETF还是龙头
2. 当前属于什么市场阶段
3. 这笔属于哪种买点
4. 核心支撑位在哪里
5. 失效位和止损位在哪里
6. 如果不对，最晚什么时候走

答不上来，就不做。

## 八、卖出规则

卖出按优先级分四层。

### 1. 第一层：破位卖

下面任何一种成立，都属于结构被证伪：

- 跌破前低
- 跌破箱体下沿
- 跌破 `40天线` 关键支撑
- 启动阳线被完全吞没

破位之后，不要用“长期看好”给自己找借口。

### 2. 第二层：时间价值耗尽

如果买入后：

- `2到5个交易日` 没走出应有强度
- 没有形成浮盈
- 一直反复磨

说明这笔交易效率不高，应该减仓或离场。

### 3. 第三层：趋势钝化

钝化的典型表现：

- 走不动
- 冲高回落变多
- 一波比一波弱
- 形成圆顶或尖顶
- 弹簧松开后又反复压紧
- 明显弱于板块

处理原则：

- ETF：先减，再看板块是否仍强
- 龙头：优先减，因为龙头回撤会更快更深

### 4. 第四层：压力位 / 利好尽出

常见压力位：

- 前高
- 箱体上沿
- 密集换手区
- 板块情绪过热区

这里不是一定清仓，但至少该把利润装一部分进口袋。

## 九、仓位规则

你这份手册是偏进取版，所以仓位会更主动，但必须有边界。

### 1. 单只仓位上限

- 单只ETF常规上限：`20%到30%`
- 单只龙头常规上限：`15%到25%`

### 2. 合计仓位上限

- 启动期龙头合计仓位：`30%到50%`
- 主升期龙头合计仓位：`40%到70%`
- 总仓位高点：可以很高，但建议仍保留 `10%到20%` 机动仓

### 3. 建仓方式

默认采用：

- `先一笔`
- `确认后最多再加一笔`

不建议：

- 一把梭
- 连续追着加
- 越跌越买摊平

### 4. ETF和龙头的不同仓位逻辑

- ETF可以先上，因为它承担方向确认和心态稳定的作用
- 龙头只在主线明确、结构清晰时加重
- 龙头仓位的增加，必须建立在原有仓位已经走对的前提下

### 5. 每笔交易先算风险，再反推仓位

顺序必须是：

1. 先定止损位
2. 再算这笔最多亏多少钱
3. 再决定上多少仓位

不允许先拍脑袋上仓，再事后找止损。

## 十、禁止动作

这部分比买点还重要。

明确禁止：

- 在下降通道里抄底
- 远离支撑位追高
- 因为怕错过而临时改计划
- 越跌越买摊平
- 一边做趋势，一边用价值投资给自己找借口
- 高位情绪爆炸时重仓追龙头
- 原本只是波段，最后拖成长期死扛

## 十四、一页执行版

如果你以后只记得最重要的部分，就记下面这几条：

1. `先判断市场阶段，再决定ETF和龙头的重心`
2. `ETF负责方向，龙头负责弹性`
3. `只做第一次回档、二次压紧/双底、启动第一阳线`
4. `收盘做主决策，盘中只做确认和应急`
5. `先定止损，再定仓位`
6. `错了先走，做对再加`

## 十五、最后的提醒

这份手册的重点，不是让你每天都做很多交易，而是让你在有限时间里：

- 做自己能看懂的机会
- 用ETF控制方向和波动
- 用龙头在主升段提高弹性
- 把错误控制在还能继续出手的范围内

真正适合上班族的，不是最炫的打法，而是你能稳定执行三个月、六个月、一年的打法。
