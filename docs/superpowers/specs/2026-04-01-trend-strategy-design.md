## 1. 策略概述

### 1.1 目标
捕捉强势股回调后的趋势延续，结合短线爆发与中线趋势。

### 1.2 策略类型
股票趋势跟踪策略，持有周期 2天 - 3周。

---

## 2. 买入条件

### 2.1 触发信号（条件1）
- **30日涨幅 ≥ 50%**
- 计算方式：`(过去30日最高价 - 过去30日最低价) / 过去30日最低价 ≥ 0.5`
- 满足此条件后，股票进入"候选池"

### 2.2 回调确认（条件2）
- **股价进入 40日均线 ±5% 范围**
- 计算方式：`|收盘价 - MA40| / MA40 ≤ 0.05`
- 不限制回调时间，可以是几天或几周后

### 2.3 突破确认（条件3，需同时满足）
当股票在均线附近时，检测以下三个条件：

| 条件 | 要求 | 说明 |
|------|------|------|
| 涨幅 | 当日盘中涨幅 ≥ 5% | `(当日最高价 - 昨收) / 昨收 ≥ 0.05` |
| 成交量 | 成交量 ≥ 5日均量的 1.5 倍 | `当日成交量 / MA5_volume ≥ 1.5` |
| 价格突破 | 收盘价突破 5日高点 | `收盘价 > max(过去5日最高价)` |

**买入时机**：条件满足且当日盘中涨幅达到 `≥5%` 后，立即触发买入，不等待收盘。

---

## 3. 卖出条件

任一条件触发即卖出：

| 条件 | 触发规则 | 说明 |
|------|---------|------|
| 止损 | 亏损 ≥ 10% | `(买入价 - 当前价) / 买入价 ≥ 0.10` |
| 利润不达标 | 买入后2个交易日利润 < 3% | 短线筛选，不符合预期则离场 |
| 单日大跌 | 当日跌幅 ≥ 5% | `(昨收 - 当前价) / 昨收 ≥ 0.05` |
| 其他 | - | 继续持有 |

---

## 4. 仓位管理

- **每次买入**：总资金的 20%
- **最大持仓**：可同时持有 5 只股票
- **资金不足时**：跳过该信号
