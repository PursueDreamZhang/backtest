# 趋势投资买卖逻辑总结

基于以下四份材料整理：

- 《毛选》中竟藏了一套顶级趋势交易系统？
- 首次公开！职业级短线交易策略
- 弹簧策略思考1
- 弹簧策略思考2

这套方法的核心，不是去预测最低点和最高点，而是用“运动战 + 弹簧 + 赔率思维”去做趋势交易：在有利战场低成本试错，做对了就扩大胜利，做错了就及时撤退。

## 一、这套方法的核心思想

### 1. 先有趋势，后有弹簧

弹簧只是买点，不是趋势本身。先要看到市场已经开始选择方向，或者某个板块已经表现出强势，再去等它第一次回档、二次压紧、双底确认这类低风险切入点。

换句话说，这套方法不是无脑抄底，而是在趋势已经有苗头、甚至已经被市场选出来之后，寻找更舒服的上车位置。

### 2. 靠赔率赚钱，不靠高胜率赚钱

作者反复强调，这种打法未必能显著提高胜率，很多时候胜率也就是五五开，但只要买点足够靠近支撑、止损足够果断、做对时愿意拿住，整体结果仍然可以很好。

本质是：

- 错的时候亏得小
- 对的时候赚得大
- 不让一次错误拖成长期被套

### 3. 运动战的重点是保留主动权

这套方法最怕的是本来只是一次试错，最后被拖成“阵地战”。

因此核心要求不是“每次都判断对”，而是：

- 看错了就撤
- 趋势走顺了就拿
- 永远让自己还有下一次出手的能力

## 二、买入前先判断有没有参与资格

不是所有下跌都叫弹簧压紧，也不是所有回调都值得接。

真正适合参与的前提通常有下面几个：

### 1. 不能已经进入明确下降通道

文档里对这一点讲得很明确：

- 弹簧压紧，是“快要破位但还没破位”
- 如果已经跌破关键支撑并进入下降通道，就属于弹簧断掉了

也就是说，最好的机会不是“已经烂掉以后去捡”，而是“临近关键支撑、市场还可能把它救回来”的那一下。

### 2. 市场已经开始选择方向

作者一再强调，主线不是自己幻想出来的，而是市场选出来的。

因此更合理的顺序是：

1. 先确认板块、指数或个股所在方向正在被资金关注
2. 再等待回档到支撑位附近的低风险买点

不是先埋伏一个自己喜欢的方向，再等市场来证明你对。

### 3. 价格已经回到有利战场

这套方法非常重视“战场位置”。典型的有利位置包括：

- 20天线或40天线附近
- 一段震荡区间的下沿
- 前低附近
- 双底右侧的确认位置
- 长时间整理后的启动位

离支撑越近，试错成本越低，赔率越好。

### 4. 最好伴随缩量、波动率收敛或情绪出清

文档里多次提到，好的回档往往不是放量崩，而是：

- 成交量下降
- 波动率收窄
- 市场已经没什么人愿意继续卖
- 情绪很差，但很难更差

这说明抛压在衰减，反转或再启动更容易出现。

## 三、具体买入逻辑

### 1. 最重要的买点：弹簧压紧后刚松开

作者最推崇的买点是：

- 价格跌到关键支撑附近
- 再跌已经比较困难
- 随后开始反弹、反包、转强

也就是“弹簧压到最紧，刚开始松开”的瞬间。

原因很简单：

- 这个位置离止损最近
- 这个位置对错最容易验证
- 这个位置如果走对，后面往往是一整段趋势

### 2. 最常见的场景：启动后的第一次回档

这套方法最喜欢的不是高位追涨，而是：

- 一个方向已经启动
- 第一次明显回档
- 回到均线、前低、箱体下沿等支撑附近
- 跌不动后重新转强

作者把它叫做“以震荡的成本博一把趋势”。

### 3. 也接受少数“刚启动就追”的情况

如果没买到压紧点，还有一种例外情况可以参与：

- 长时间整理后
- 突然出现第一根启动大阳线
- 板块或个股处在趋势刚刚展开的阶段

这种时候也可以追，但要接受两个事实：

- 仓位不能太重
- 止损要更严格

因为此时赔率已经不如支撑位附近那么舒服。

### 4. 典型买入信号清单

结合四份材料，常见的买入信号可以整理成下面这些：

| 信号 | 含义 |
| --- | --- |
| 回到20天线/40天线附近止跌 | 回到平均成本区，容易形成支撑 |
| 回到震荡区间下沿不破 | 说明箱体仍有效 |
| 回到前低附近不破 | 说明低点经受住压力测试 |
| 双底确认或二次压紧 | 支撑更明确，赔率更高 |
| 缩量回调后重新放量上行 | 抛压减弱，新的力量进场 |
| 情绪极差但没有继续恶化 | 利空接近出尽，局面难再变坏 |
| 事件催化出现，预期开始形成 | 原先平衡被打破，趋势可能被放大 |

### 5. 买入时必须同时想清楚的五件事

这套方法不是“看到像就冲”，而是强调事前定义：

1. 支撑位在哪里
2. 止损位在哪里
3. 这笔交易的时间窗口有多长
4. 如果走对，什么情况下允许继续拿
5. 如果走错，什么情况下必须撤

如果这五件事没想清楚，买入后大概率会被盘中波动带着走。

## 四、仓位与加仓逻辑

### 1. 以损定量，而不是以感觉定仓

作者非常强调“以损定量”。

先想清楚：

- 这笔交易最多愿意亏总资金的多少
- 止损位距离当前买点多远

再反推出可以下多少仓位。

这意味着：

- 仓位大小取决于风险，不取决于冲动
- 止损太远的机会，即使看起来很诱人，也可能不值得做

### 2. 最多加仓一次，且要有补偿

加仓不是因为“想赚更多”，而是因为：

- 概率提高了
- 或者赔率又变好了

常见加仓场景：

- 第一笔试错成功后，回档但不破，再次转强
- 形成双底、二次压紧
- 趋势已经带来浮盈，新的回档点又很舒服

作者多次提到，一般最多加仓一次，不建议一路越跌越买。

### 3. 明确反对在下降通道里摊平

这套方法和均值回归式补仓最大的区别，就是不愿意在明显错误的趋势里不断补仓。

原因是：

- 越跌越买容易把运动战打成阵地战
- 一旦市场不均值回归，会非常被动
- 心理压力也会越来越大

## 五、持有逻辑

### 1. 如果趋势是对的，通常应该很快对起来

文档里有一句反复出现的话：

“如果趋势是对的，就应该立刻对。”

它的意思不是要求买完马上暴涨，而是说：

- 至少不应该买完后一直磨
- 不应该反复回踩、反复压紧
- 不应该长时间走不动

一旦买入后表现明显低于预期，就要重新评估。

### 2. 给正常趋势时间，不给错误趋势时间

这套方法承认趋势需要发酵时间，但这个时间只给“正常趋势”。

如果是短线试错，在没有浮盈的前提下，作者通常不会让交易拖太久，常见表述是：

- 不超过一个隔夜
- 或者两个交易日内要见分晓

如果一直没有走顺，说明时间价值在流失，继续拿着并不划算。

### 3. 对强趋势要敢于持有

作者并不主张一有利润就跑。

真正的强趋势往往会不断自我强化，因此在趋势没有明显变坏之前，应该让利润继续奔跑，而不是总想着卖在最高点。

## 六、卖出逻辑

卖出逻辑可以归纳为五类。

### 1. 结构破坏：跌破止损位或关键支撑

这是最基础、最明确的一类卖点：

- 跌破均线支撑
- 跌破箱体下沿
- 跌破前低
- 双底失效

一旦这些位置被有效跌破，原先的买入逻辑就被证伪了，应该撤退。

### 2. 时间价值耗尽

如果一笔交易买入后：

- 没有浮盈
- 没有快速走强
- 一直在原地磨

那么就说明这笔交易效率很低，应该及时离场，把主动权拿回来。

### 3. 趋势钝化或异常反向波动

这是作者非常看重的一类卖点。

所谓“趋势钝化”，可以理解为：

- 走势明显走不动了
- 上攻力度一波比一波弱
- 形成圆顶、尖顶
- 本来该强，结果突然来了一根异常大阴线
- 多次碰压力位都过不去
- 弹簧松开后又反复压紧

一句话总结就是：

如果这真是一段正常上涨趋势，它不应该长成这样。

### 4. 到达压力位且没有新的力量突破

典型压力位包括：

- 震荡区间上沿
- 前高
- 240分钟级别的重要阻力
- 前期密集换手区

如果到这里明显走不动，而且距离下一次催化还很远，就可以考虑先带钱走。

### 5. 利好尽出，情绪难以更好

这是一种博弈性卖点。

当市场已经把好消息都交易完了，情绪来到极热状态，接下来很难再有更好的边际变化，就要开始准备撤退。

相关辅助信号还包括：

- 顶部区域累计换手率过高
- 热点高度一致
- 板块已经从主升进入鱼尾

## 七、哪些情况宁可错过也不要做

这套方法不是“所有机会都做”，相反，它非常强调舍得错过。

以下情况通常不值得强做：

- 已经远离支撑位，还想硬追
- 已经进入下降通道，还想抄底
- 看不清支撑和止损在哪里
- 市场没有选出方向，只是自己主观觉得会涨
- 错过最佳买点后，为了不甘心去高位补票

作者的思路很明确：错过不可惜，做错且不肯走才最伤。

## 八、一句话总结

这套趋势投资方法，可以浓缩成一句话：

在市场已经选出的强势方向里，等待价格回到关键支撑位附近，以小成本试错；弹起来就让利润奔跑，弹不起来就立刻撤退。
