# 趋势跟踪策略逻辑说明

本文档用于说明项目当前默认趋势跟踪策略的交易逻辑，重点描述策略的买入条件、卖出条件、仓位控制方式以及整体思路，便于从业务角度理解策略行为。

## 策略概览

该策略的核心思路不是简单追涨，而是尝试在强势股完成一轮拉升之后，等待其回调到关键支撑区域，再观察是否出现缩量企稳和重新向上突破的信号，随后以受控仓位介入。如果买入后走势不够强，则尽快退出；如果走势延续，则尽量让利润继续扩张，但在趋势转弱时及时止盈离场。

从交易风格上看，这是一套偏向“强势股二次启动捕捉”的趋势策略。

## 买入逻辑

策略的买入逻辑分成多个阶段，只有全部条件同时满足时才会触发买入。

### 1. 强势股识别

策略会先观察最近一段时间内的价格表现，判断该股票是否曾经出现过明显上涨。默认规则是：最近 30 个交易日内，区间涨幅达到或超过 50%，则将该股票视为已经完成过一轮强势拉升。

### 2. 从高点回落

被识别为强势股之后，策略不会立即追入，而是继续等待其从近期高点出现一定幅度的回调。默认要求是：当前价格相对前期强势阶段高点，至少回落 5%。

### 3. 靠近支撑位

在回落之后，策略还会判断当前价格是否已经接近关键支撑区域。默认会同时参考以下几类支撑：

- 中短期均线支撑
- 主趋势均线支撑
- 近期箱体下沿支撑

### 4. 缩量调整

策略会继续判断回调过程中的成交量是否出现明显收缩。默认规则是：短期平均成交量明显低于中期平均成交量。

### 5. 再次向上突破

在确认强势背景、回调幅度、支撑位置和缩量状态后，策略还需要一个重新转强的触发信号。默认规则是：当日盘中价格相对前一交易日收盘价上涨达到 5% 或以上，即视为突破触发。

## 卖出逻辑

买入之后，策略会持续监控持仓表现，一旦满足任一卖出条件，就执行离场。卖出条件按风险优先、效率优先、趋势优先的思路设计。

## 策略特点

- 它不是抄底策略，而是强势延续策略
- 它不是纯追涨策略，而是等待回调后的再启动
- 它强调支撑、缩量和突破三个条件的组合
- 它对买入后的表现有较高要求，不愿意长期持有弱势仓位
- 它用仓位控制和多层退出机制来限制单笔交易风险
